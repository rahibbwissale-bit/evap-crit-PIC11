# Projet PIC — Évaporation multiple & Cristallisation du saccharose

Application pédagogique développée dans le cadre du module **Évaporation et Cristallisation** (Filière PIC), permettant de visualiser et d’illustrer les principes d’**évaporation multiple** et de **cristallisation du saccharose** via une interface web interactive (Streamlit).

## Démo en ligne

Une version en ligne de l’application est disponible :

https://evap-crit-pic11-v6hyd7a2cyovj45r7qh66x.streamlit.app/
## Installation rapide

Prérequis :
- Python 3 installé
- `pip` disponible dans le terminal

Cloner le projet et se placer dans le dossier :
https://github.com/rahibbwissale-bit/evap-crit-PIC11.git

## Exécution locale
Installer les dépendances : 
- pip install -r requirements.txt
- streamlit run streamlit_app.py

## À propos

Ce projet a pour but de soutenir la compréhension des phénomènes d’évaporation multiple et de cristallisation du saccharose dans un contexte de génie des procédés, à travers une interface simple, accessible et interactive.
